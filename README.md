# QTDA Project

## Cấu trúc dự án

```
QTDA/
├── index.html          # Trang chủ
├── css/
│   └── styles.css      # Style tùy chỉnh
├── js/
│   ├── main.js         # File JavaScript chính
│   └── utils.js        # Các hàm tiện ích
├── assets/
│   ├── images/         # Ảnh
│   ├── fonts/          # Font chữ
│   └── icons/          # Icon
├── .gitignore
└── README.md           # File này
```

## Hướng dẫn sử dụng

### Khởi động dự án

1. Mở file `index.html` trong trình duyệt
2. Hoặc sử dụng Live Server (VS Code Extension)

### Cấu trúc thư mục

- **css/** - Chứa các file CSS tùy chỉnh
- **js/** - Chứa các file JavaScript
- **assets/** - Chứa các tài nguyên như ảnh, icon, font

## Công nghệ sử dụng

- Tailwind CSS - Styling framework
- Material Icons - Icon library
- Vanilla JavaScript

## Notes

- Tailwind CSS được load từ CDN trong `index.html`
- Hỗ trợ Dark Mode
- Responsive Design
